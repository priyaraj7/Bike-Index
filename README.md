## Description:

This project is built using npm and Bike-index url.

## Setup/Installation Requirements

- Open the Terminal
- Clone the repository
- Navigate to the directory
- Install npm

```
$ npm install
```

- Build a package

```
$ npm run build
```

- Usage

```
$ npm run start
```
